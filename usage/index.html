Usage | RevFrame

This framework will help in scaling Machine Learning/Deep Learning/Artificial Intelligence/Natural Language Processing Models to Web Application level almost without any time.

How to Use

  • Clone revframe - github link using git clone https://github.com/revanth-reddy/revframe.git
  • cd revframe - Main Programming Language for this Framework
  • docker compose up - To build and run the revframe.
  • visit http://127.0.0.1:8000/ to see the application running.
  • At http://127.0.0.1:8000/model you can input a sample word to generate a text and it can be seen in the output.

How to use Custom Model or add your model to RevFrame

Now build the containers and restart using:

  • docker compose build
  • docker compose up
